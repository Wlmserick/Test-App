## Setup

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd <repository-folder>
   ```

2. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

3. **Set Up Google Maps API Key**:
   - Add your API key to `AndroidManifest.xml` (for Android) and `AppDelegate.swift` (for iOS).

4. **Run the App**:
   ```bash
   flutter run
   ```
